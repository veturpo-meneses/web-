Inicializa el proyecto con npm init y asegúrate de que el archivo de entrada sea server.js.
Instala Express:

npm install express
node server.js


Para probar el código:

Navega a http://localhost:8080/initialize para inicializar los datos.
Usa rutas como http://localhost:8080/countries para obtener todos los países, 
http://localhost:8080/countries/124 para obtener detalles de Canadá, y así sucesivamente.
